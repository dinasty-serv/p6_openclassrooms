# p6_openclassrooms
Installation: 
1) git@github.com:dinasty-serv/p6_openclassrooms.git ./snowtricks
2) cd snowtricks
3) composer install
4) php bin/console fixtures:load
5) php bin/console server:start 0.0.0.0:8000
